# 使い方

`python3 generate_plist.py` を実行すると `user-dictionary.plist` が生成されるので、それを以下の場所にドラッグする。

> 「ユーザ辞書.plist」ファイルを読み込むには、日本語入力ソースの使用中に、ファイルを「キーボード」設定の「ユーザ辞書」ダイアログにドラッグします。

<https://support.apple.com/ja-jp/guide/japanese-input-method/jpim10228/mac>
